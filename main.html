JavaScript — Bringing Life to Web

JavaScript is the programming language of the web, adding interactivity, dynamic behaviour, and powerful functionality to websites and applications.

Interactive Dynamic Event-Driven Versatile

HTML

HTML is the structure of a web page. It defines headings, paragraphs, images, links, and other content.

What it is

HTML stands for HyperText Markup Language. It gives web pages their basic layout and content.

How it works

Browsers read HTML tags and turn them into visible elements such as buttons, text, and images.

Example
<h1>Hello World</h1>
<p>This is a paragraph.</p>
<a href="https://example.com">Visit</a>

CSS

CSS is used to style web pages by controlling colours, spacing, layouts, and typography.

What it is

CSS stands for Cascading Style Sheets. It makes websites look attractive and organised.

How it works

CSS targets HTML elements and applies visual rules like font size, background colour, and margins.

Example
h1 {
  color: gold;
  text-align: center;
}

JavaScript Fundamentals

Learn the core concepts that make JavaScript the language of modern web development.

What it is

JavaScript is the programming language that makes web pages interactive by responding to clicks, input, and events.

How it works

JavaScript runs in the browser, reads your code, and changes the page dynamically based on user actions.

Example
const name = 'Asha';
console.log(`Hello, ${name}!`);
Variables & Functions
// Variables
let name = "Sanjog Thapa";
const age = 22;
var course = "CET138";

// Function
function greetStudent(studentName) {
    return `Hello, ${studentName}! Welcome.`;
}

// Arrow Function
const calculateGrade = (score) => {
    return score >= 90 ? 'A' : score >= 80 ? 'B' : 'C';
};
Arrays & Objects
// Array
const technologies = ['HTML', 'CSS', 'JS'];

// Object
const student = {
    name: 'Sanjog Thapa',
    college: 'ISMT College',
    course: 'CET138 Full Stack',
    skills: technologies,
    introduce: function() {
        return `I'm ${this.name} from ${this.college}`;
    }
};

Element Selection

Creating Elements
// Create new element
const newDiv = document.createElement('div');
newDiv.className = 'alert alert-success';
newDiv.innerHTML = '<strong>Success!</strong>';

// Add to page
document.body.appendChild(newDiv);

// Remove after 3s
setTimeout(() => newDiv.remove(), 3000);

Event Handling

Make your websites interactive by responding to user actions.

Click Events
// Add event listener
button.addEventListener('click', function() {
    alert('Button clicked!');
});

button.addEventListener('mouseenter', () => {
    button.style.backgroundColor = 'red';
});

button.addEventListener('mouseleave', () => {
    button.style.backgroundColor = '';
});

Form Events
// Form validation
form.addEventListener('submit', (e) => {
    e.preventDefault();
    const name = document.getElementById('name').value;
    if (name.length < 2) {
        alert('Name too short!');
        return;
    }
    alert('Form submitted!');
});
